## 🎮 게임 시나리오

1. 게임 시작
    - 자동차 이름 입력 문구를 출력한다
         ```
         경주할 자동차 이름을 입력하세요.(이름은 쉼표(,) 기준으로 구분)
         ```
    - 자동차 이름을 입력받는다.
    - 시도할 횟수를 묻는다.
        ```
         시도할 회수는 몇회인가요?
         ```
    - 횟수를 입력받는다.
    - 사용자가 잘못된 값을 입력할 경우 IllegalArgumentException을 발생시킨 후 애플리케이션은 종료되어야 한다.
2. 게임 진행
    - 0에서 9 사이의 무작위 값을 구한 후 무작위 값이 4 이상일 경우 전진한다.
    - 각 차수별 실행 결과를 출력한다.
   
3. 게임 종료
    - 누가 우승했는지를 알려준다. 이때 우승자는 한명 이상일 수 있고, 여러명일 경우 쉼표를 이용해 구분한다.
